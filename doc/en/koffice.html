KOffice


Introduction

KOffice is an Office suite for the K Desktop Environment. It consists different productivity apps:

  • KPresenter
  • KSpread
  • KChart
  • KIllustrator
  • Kimage
  • KFormula
  • KWord

Other apps like KLyx will be part of KOffice in the near future.

Its development is based on the OpenParts/KOM model developed by Torben Weis.
The OP technique allows to embed other KOffice components in each other, e.g. you  can make a presentation with KPresenter and embed for example a KChart while running KPresenter.

KOffice uses standardized actions for selecting and copying text. These standardized actions are described in the misc section.
