KPresenter Help
 
 

The File Menu KPresenter

The file menu consists of all actions to create, open and save new presentation documents.
 

New

This command creates a new document. You are presented with the Choose a Template dialog.
Open...
You created and saved any kpresenter document before? Or you got any other kpresenter document and like to view it? To open a kpresenter document use this!

Save

To save the active document use this command. If your active document has no name assigned you are presented with the save as... dialogue.

Save as...

You like to save your document and assign a new name for it?

Close

This closes the active document.

Quit

Use this command to close kpresenter.
