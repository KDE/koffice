ReportHeader & ReportFooter Sections

Description:

The ReportHeader and ReportFooter elements define report sections that are usually printed at the beginning and end of the report.

Definitions:

<!ELEMENT ReportHeader (Line*, Label*, Special*)>
<!ATTLIST ReportHeader
    Height          CDATA #REQUIRED
    PrintFrequency  CDATA #REQUIRED>

<!ELEMENT ReportFooter (Line*, Label*, Special*, CalculatedField*)>
<!ATTLIST ReportFooter
    Height         CDATA #REQUIRED
    PrintFrequency CDATA #REQUIRED>

Elements:

The ReportHeader element can contain the following elements:
  • Line objects- Line
  • Label objects - Label
  • Special objects - Special
The ReportFooter element can contain the following elements:
  • Line objects- Line
  • Label objects - Label
  • Special objects - Special
  • Calculated field objects - CalculatedField

Attributes:

Height - Sets the height of the report section.  If you don't want this section, set this value to zero.

PrintFrequency - Set the print frequency of the section.
 
Value Print Frequency
0 First Page
1 Every Page
2 Last Page
