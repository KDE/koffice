Detail Section

Description:

The Detail element defines the report section that contains the report data.

Definition:

<!ELEMENT Detail (Line*, Label*, Special*, Field*)>
<!ATTLIST Detail
    Height  CDATA   #REQUIRED>

Elements:

The Detail element can contain the following elements:
  • Line objects- Line
  • Label objects - Label
  • Special objects - Special
  • Field objects - Field

Attributes:

Height - Sets the height of the report section.
