PageHeader & PageFooter Sections

Description:

The PageHeader and PageFooter elements define report sections that are usually printed on every page of the report.

Definitions:

<!ELEMENT PageHeader (Line*, Label*, Special*)>
<!ATTLIST PageHeader
    Height          CDATA #REQUIRED
    PrintFrequency  CDATA #REQUIRED>

<!ELEMENT PageFooter (Line*, Label*, Special*)>
<!ATTLIST PageFooter
    Height          CDATA #REQUIRED
    PrintFrequency  CDATA #REQUIRED>

Elements:

The PageHeader element can contain the following elements:
  • Line objects- Line
  • Label objects - Label
  • Special objects - Special
The PageFooter element can contain the following elements:
  • Line objects- Line
  • Label objects - Label
  • Special objects - Special

Attributes:

Height - Sets the height of the report section.  If you don't want this section, set this value to zero.

PrintFrequency - Set the print frequency of the section.
 
Value Print Frequency
0 First Page
1 Every Page
2 Last Page
