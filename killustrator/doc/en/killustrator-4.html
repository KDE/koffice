4. Frequently Asked Questions

Question: How can I use EPS files, which were exported from KIllustrator, in LATEX?
Answer: This is possible using the clip option (\epsfig{file=graphics.eps, width=9cm, clip= })

Question: How can I import ASCII text into a KIllustrator document?
Answer: The importing of text is performed by copying/pasting. Copy the text from (as an example) a text editor. In KIllustrator select Paste from the File menu, and position the text as desired.
