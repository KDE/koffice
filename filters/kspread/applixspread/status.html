Import Applix Spreadsheet for kspread
Last update 03 mar 2001
Features - can import simple and complex Applix Spreadsheet documents
- tests the headline of the document
- converts <, >, &, to &lt; &gt; &amp;
- converts all applix special characters
- converts fontsize, fontcolor, horizontal and vertical alignment, bold, italic, underline
- eats long text strings (more than one row in inputfile)
- converts different tables inside of one document
- cellformat: background color, brushstyles, brushcolor
- cellframe: linewidth, linetype, linecolor

Progress report Fileformat:
The file has some different parts:
1. Header
2. Colortable
3. Fonttable
...
4. View ...
5. Cellvalues
6. End


1. Header
The header looks like that:
*BEGIN SPREADSHEETS VERSION=430/430 ENCODING=7BIT


2. Colormap:
The colormap begins with
COLORMAP

and ends with
END COLORMAP

a color line is describe like that
Text 0 c m y k 0

2.1. Text = Textstring -
2.2. 0 = NULL -
2.3. c = cyan 0-255
2.4. m = magenta 0-255
2.5. y = yellow 0-255
2.6. k = black (key) 0-255
2.7. 0 = NULL -


Color converting is very simple cmyk to rgb:
r = 255 - (c+k)
g = 255 - (m+k)
b = 255 - (y+k)
if the value of r, g or b if less then 0 you must set it to 0

for examples inside koffice look a this files too
koffice/kimageshop/core/kis_color.cc and kis_color.h


5. Cellvalue
This is a cellvalue line
('DN|B,I,U,TF1,P18|T4) A!D11: Hello everybody

a cellvalue line has 3 parts:
5.1. Formatpart (*)
5.2. Location part: A!D11 : A=Table D11=Cellnumber: column:D - row:11
5.3. Info String: Hello everybody
5.4. Formatsubparts: (Part1|Part2|Part3)


Formatsubparts
There are 3 Formatsubparts
5.1.1. Form
5.1.2. Char
5.1.3. Cell


5.1.1. Cellvalue Part1:


5.1.2. Cellvalue Part2:


5.1.2.1. B = bold
5.1.2.2. I = italic
5.1.2.3. U = underline
5.1.2.4. TFx = fontfamily. x is the number in the fonttable
5.1.2.5. Px = fontsize. x is the fontsize


Cellvalue Part3:
< 5.1.3.1. Txy = Frame at top
5.1.3.2. Bxy = Frame at bottom
5.1.3.3. Lxy = Frame at left side
5.1.3.4. Rxy = Frame at right side

y: example FG7 SH11FG18 SH11FG18BG4
it has max 3 Parts:
5.1.3.0.1. FGx = Foreground color. x is the number in the colortabl
5.1.3.0.2. SHx = Background shade type. x is the shadetypenumber
5.1.3.0.3. BGx = Background color. x is the number in the colortable


3. Fonttable:
The colormap begins with
TYPEFACE TABLE
and ends with
END TYPEFACE TABLE
Helvetica


4. View:
Each table has its own view section:
If there is a table with the name A then it starts with:
View Start, Name: ~A:~
and it ends with: View End, Name: ~A:~

In the view of an table block is for example the width of a column:
4.1. View Column Widths: A:3 B:3 C:3 D:3 E:3 F:3 G:4 H:4
or the heigth of a row
4.2. View Row Heights: 3:18 4:24 8:32804
6. End
The end looks like that:
*END SPREADSHEETS
