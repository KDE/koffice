Import AbiWord for KWord
Last update 12 Jun 2005
Features - Import text
- Can import formatting
- Can read gzip-compressed files (just use .abw.gz or .zabw as an extension of the filename!)
- Can read bzip2-compressed files (just use .abw.bz2 or .bzabw as an extension of the filename!)
- Styles
- Result of character styles
- Images (PNG)
- Fields (as far as KWord supports them.)
- Page size, including page margins
- Import ignored word list (for spell checking)
Todo - Multiple columns
- Lists
- Header/footer
- Bookmarks
- Tables
- Footnotes
History 19 Jan 2001: First half working import filter.
24 Jan 2001: Can read gzip compressed AbiWord files.
31 Jan 2001: Can import all formating at character level.
09 Jan 2002: Can import styles.
27 Feb 2002: Fields.
19 Mar 2002: More styles, inlcuding character ones.
07 Jun 2002: Ignored Word List
28 Jan 2003: new XML parsing error message box
12 Jun 2005: import keyword/subject in documentinfo

Export KWord to AbiWord
Last update 25 Jan 2003
Features - Can export simple text
- Can export formatting
- Can write gzip-compressed files (just use .abw.gz or .zabw as an extension of the filename!)
- Can handle forced page breaks
- Can export inlined pictures (converted to PNG)
- Can export styles
- Can export variables into fields (no custom variables!)
- Can export page size, including page margins
- Can export the ignored word list (for spell checking)
Todo - Multiple columns.
- Lists.
- Inlined formula (MATHML)
- Non-inlined pictures
- Tables
- KWord's DTP modus (How can you export that in AbiWord?)
- Bookmarks
- Footnotes
- Many other things...
