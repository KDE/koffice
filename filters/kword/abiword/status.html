Import AbiWord for KWord
Last update 31 Jan 2001
Features - Import text.
- Can export simple formating (font name, font size, bold, italic, underline, colour, subscript superscript).
- Can read gzip-compressed files (just use .abw.gz or .zabw as an extension of the filename!)
Todo - Multiple columns
- Styles
- Images (PNG)
- Stop KWord from crashing when importing wrong documents.
- Many other things...
History 19 Jan 2001: First half working import filter.
24 Jan 2001: Can read gzip compressed AbiWord files.
31 Jan 2001: Can import all formating at character level.
Authors Nicolas GOUTTE

Export KWord to AbiWord
Last update 31 Jan 2001
Features - Can export simple text.
- Can export simple formating (font name, font size, bold, italic, underline, colour, subscript, superscript).
- Can write gzip-compressed files (just use .abw.gz or .zabw as an extension of the filename!)
Todo - Multiple columns.

- Many other things...
