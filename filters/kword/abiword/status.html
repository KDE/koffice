Import AbiWord for KWord
Last update 16 Aug 2001
Features - Import text.
- Can export simple formating (font name, font size, bold, italic, underline, colour, subscript superscript, text background color, right magin,left margin, first line spacing, line spacing, tabulator).
- Can read gzip-compressed files (just use .abw.gz or .zabw as an extension of the filename!)
Todo - Multiple columns
- Styles
- Lists
- Images (PNG, SVG...)
- Many other things...
History 19 Jan 2001: First half working import filter.
24 Jan 2001: Can read gzip compressed AbiWord files.
31 Jan 2001: Can import all formating at character level.

Export KWord to AbiWord
Last update 29 Nov 2001
Features - Can export simple text.
- Can export simple formating (font name, font size, bold, italic, underline, colour, subscript, superscript, text background color, line spacing, tabulator).
- Can write gzip-compressed files (just use .abw.gz or .zabw as an extension of the filename!)
- Can handle forced page breaks
- Can export inlined PNG images (JPEG should also work, if AbiWord can read JPEG.)
- Can export styles
Todo - Multiple columns.
- Lists.
- Other inlined images (SVG, MATHML)
- Non-inlined images
- Transform tables into something AbiWord can understand.
- KWord's DTP modus (How can you export that in AbiWord?)
- Many other things...
History 17 Jan 2001: First filter version (can export simple texts and formats)
21 Jan 2001: Added colours and corrected bug in font size
31 Jan 2001: New form of comment header in the exported file
19 Mar 2001: Fix the bug due to the new KWord
16 Aug 2001: Fix forced page break handling (only KOffice 1.2)
29 Nov 2001: Export of inlined images (PNG) (only KOffice 1.2)
