WordPerfect Filter for KWord

Ariya Hidayat (ariya at kde.org)
Last update: 17 August 2002

This is WordPerfect filter for KWord, which lets KWord imports and exports to WordPerfect document format. Currently, supported WordPerfect document is version 5.1, 5.2, 6, 7, or later (this includes the latest WordPerfect 2002).

The filter itself is (always) in development stage so not all WordPerfect features are already supported. However, also expect some improvements which somehow may (or may not) happen over the time. The feature matrix provided below will help the users to track which features are supported and which aren't.

For a complete reference on WordPerfect 7 file format, please visit Corel WordPerfect 7 file format (at Corel website). In addition, WordPerfect 5.1 format is also available (at wotsit.org). Although the description of the file format is rather clear and concise, the implementation for a few parts can be hard and tricky.

Comments, helps, and patches are all welcomed.
